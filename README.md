# CV Builder Application

A modern CV/Resume builder application built with React that allows users to create professional resumes with multiple templates and download them as PDFs.

## Features

- **Multiple Templates**: Choose from various professional CV templates
- **PDF Export**: Download your CV as a high-quality PDF
- **Payment Integration**: Secure payment processing for CV downloads
- **Admin Panel**: Secure admin interface for managing payments and user data
- **Responsive Design**: Works on desktop and mobile devices

## Admin Panel Security

The admin panel has been secured with the following features:

- **Hidden Access**: Admin credentials are no longer visible in the UI
- **Keyboard Shortcut**: Press `Ctrl + Alt + A` to reveal admin toggle
- **Rate Limiting**: Maximum 3 failed login attempts
- **Environment Variables**: Credentials stored securely in `.env` file
- **Access Control**: Admin toggle is hidden by default

### Setting Up Admin Access

1. Run `npm run setup-admin` to set up your admin credentials
2. Restart the development server
3. Press `Ctrl + Alt + A` to reveal the admin toggle
4. Use your credentials to access the admin panel

For detailed security information, see [ADMIN_SECURITY_SETUP.md](./ADMIN_SECURITY_SETUP.md).
