Create Shopkeeper Table in Supabase

Instructions:

  1. Go to your Supabase Dashboard
  2. Navigate to the SQL Editor
  3. Copy the SQL code below
  4. Paste it into the SQL Editor
  5. Click "Run" to execute the SQL

SQL Code to Create the Table:

CREATE TABLE public.shopkeeper_cvs ( id SERIAL PRIMARY KEY, shopkeeper_id UUID NOT NULL, shop_name VARCHAR(255) NOT NULL, cv_name VARCHAR(255), name VARCHAR(255), email VARCHAR(255), phone TEXT, address TEXT, objective TEXT, image_url TEXT, education JSONB DEFAULT '[]', work_experience JSONB DEFAULT '[]', skills JSONB DEFAULT '[]', certifications JSONB DEFAULT '[]', projects JSONB DEFAULT '[]', languages JSONB DEFAULT '[]', hobbies JSONB DEFAULT '[]', cv_references JSONB DEFAULT '[]', other_information JSONB DEFAULT '[]', custom_sections JSONB DEFAULT '[]', template VARCHAR(50) DEFAULT 'classic', created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(), updated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW() ); -- Enable Row Level Security ALTER TABLE public.shopkeeper_cvs ENABLE ROW LEVEL SECURITY; -- Create policy for shopkeepers to access their own CVs CREATE POLICY "shopkeeper_cvs_policy" ON public.shopkeeper_cvs FOR ALL USING (shopkeeper_id = auth.uid());

After Creating the Table:

Once you've created the table in Supabase, your CV saving should work perfectly! The table will store all shopkeeper CVs with proper security policies.
